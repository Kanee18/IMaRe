##  Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `F11` | Toggle Fullscreen |
| `Ctrl + Z` | Undo (Drawing) |
| `Ctrl + Y` | Redo (Drawing) |
| `Delete` / `Backspace` | Delete Selected Image |
| `Alt + Left Click` | Pan Canvas |
| `Ctrl + Scroll` | Zoom In/Out |
| `Scroll` | Pan Canvas |
